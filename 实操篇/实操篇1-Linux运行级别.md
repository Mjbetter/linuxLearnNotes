# Linux运行级别



#### 一、基本介绍

​	0：关机

​	1：单用户【找回丢失密码】

​	2：多用户状态没有网络服务

​	3：多用户状态有网络服务

​	4：系统未使用保留给用户

​	5：图形界面

​	6：系统重启

​	常用运行级别是3和5，也可以指定默认运行级别

#### 二、指定运行级别

```  
init 运行级别  --指定运行级别
```

​	一般指定默认级别是5，图形界面。

​	但工作中一般指定3这个运行级别。

```
systemctl get-default --查看当前运行级别
systemctl set-default graphical.target --设置默认运行级别（第五个级别，图形）
```
